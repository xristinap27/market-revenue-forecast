 Прогноз товарообігу ринку України

## Мета
Спрогнозувати динаміку товарообігу ринку на 12 місяців вперед 
та визначити % покриття ринку торговою мережею.

## Файли
- `market_revenue_dataset.csv` — датасет за 2024 рік
- `market_forecast.ipynb` — Jupyter Notebook з моделлю Prophet

## Модель
Prophet (Meta) — прогноз на 12 місяців по 5 категоріях товарів

## Метрики
MAPE, MAE, RMSE, R²
